# 八字 + 紫微斗数 双系统命理 Skill

## 一、本 skill 是什么

基于中华传统命理学经典文献的排盘与论命系统。通读 30+ 部典籍后提炼出的实务方法论（其中 17 部公共领域典籍随包提供全文，当代著作的方法论以通读笔记形式呈现）：

- **八字四家投票法**：徐乐吾（格局调候）+ 梁湘润（三轨+金不换+四条规则）+ 袁树珊（十六字法）+ 韦千里（八步法）四家并行，投票定共识
- **紫微斗数中州派**：陆斌兆讲义 + 王亭之注释，原典征验
- **双系统交叉验证**：八字-紫微在财运/事业/婚姻/迁移/健康/贵人/黄金期/最差期八维度上互相校验

## 二、最重要的铁律

### 铁律 1：禁止脚本排盘

所有排盘必须手动推演。脚本排盘曾出现的严重错误：
- 巳丑误判半合金（实际不带四正位，不算半合）
- 寅戌误判半合火（同样不算）
- 漏判命宫天干合
- 流月评级两级反转

**手动排盘逐步推理，每一步可检查，不会出现系统性偏差。**

唯一允许计算器辅助的：日柱查万年历（纯数学计算）、节气日期查询。

### 铁律 2：禁止反推（陈素庵）

> "非稽其已往，无以测其将来。"

必须先用已知事实验证框架，验证不通的不能硬解释。**禁止拿事实去凑八字。**

### 铁律 3：禁止蒙猜

不知道就说不知道。命理学里"可能""大概""应该是"都是禁词，要么有典籍出处，要么有锚点验证，要么承认不知道。

### 铁律 4：禁止截看大运

任铁樵：一运看十年，切勿上下截看。陈素庵反对"上五年看天干，下五年看地支"。**先通观十年整体性质，再"着重"前后五年，不是"只看"前五或后五。**

### 铁律 5：禁止唯一论

不能只用一个视角断命：
- 唯格局论：沈孝瞻"木逢冬水，虽透官星，亦难准贵"
- 唯神煞论：张神峰"此只把一字以定生死，实谬说也"
- 唯用神论：陈素庵"体不成则用无所附"
- 唯身强弱论：梁湘润"日主强弱是入门之优先，不是子平法全部涵义的第一优先"

## 三、八字十五步排盘流程

详见 `core/methodology.md`，以下是**必须逐步执行并逐步输出**的完整步骤：

1. **编码**：阳历 → 真太阳时（经度差+均时差）→ 四柱+藏干（本气/中气/余气）→ 大运（阳男阴女顺行，阴男阳女逆行，三日折一年）→ 起运年龄 → 流年干支
2. **识日主**：确认日干十天干，按《滴天髓》十干论把握喻象/性质/喜忌，区分阳干阴干
3. **观月令**：月支对应节气/季节，查分日用事，月令本气是否透干？季月（辰未戌丑）注意分日用事区间
4. **日主强弱**：标出所有五行，分同方/异方，结合月令加权。注意"衰而强"和"旺而弱"
5. **调候**：穷通宝鉴十干配十二月，查命局中是否有调候之神（透干/藏支/有根）。调候为急时压过格局
6. **格局**：取格优先级 = 月支本气透 > 中余气透 > 月干坐根 > 三合三会。找相神，判成败，评高低
7. **形象**：判断命局类型（独象/偶象/从象/化象/全象），判清浊（配置纯粹=清，官杀混杂=浊），辨真假（得时秉令=真），看源流（从哪起、到哪住），查通关
8. **用神**：格局/通关/病药/专旺/调候五种，不可混用。确定用神/喜神/忌神/仇神/闲神
9. **性情**：日干五行禀性 + 十神突出点（官重/杀重/印重/食重/伤重/财重/比劫重）→ 综合个性
10. **疾病**：五行受克受伤 → "取伤重者而断之"。天干对应脏腑（甲胆乙肝丙小肠丁心戊胃己脾庚大肠辛肺壬膀胱癸肾），五行太旺太弱俱病
11. **六亲**：宫位+十神参合。年=祖上，月=父母兄弟，日支=配偶，时=子女。"妻以配身，子以为后"
12. **财官**：官星状态（社会地位/事业/权力），财星状态（财富），财官印食配合
13. **大运流年**：起运计算 → 大运干支五行 → 与原局作用（用神/忌神/格局/调候/刑冲会合）→ 流年分析（表象/实质/岁运关系）→ 断事象
14. **神煞辅助**：天乙贵人、天月二德、文昌、驿马、华盖、桃花、羊刃、空亡。陈素庵"真者精择而存之，妄者悉举而削之"。神煞只是辅助信号，不能作为主要断法
15. **综合判断**：不同视角结论是否一致？矛盾处如何权衡？**必须验证已知事实**（"非稽其已往，无以测其将来"）。参以山川风土。总评格局高低

补充步骤（必须执行）：
- **A. 命宫**：地支 = 14 - 月支序 - 时支序（>12 减 12，=0 取丑）。天干用五虎遁从年干起寅顺数至命宫地支。命宫补四柱之缺/藏忌神/参与刑冲合会
- **B. 小限**：1岁小限 = 命宫干支，逐年逆行。十六字法 = 四柱8字 + 命宫2字 + 小限2字 + 大运2字 + 流年2字
- **C. 空亡**：从日柱往回数到旬头，缺的两个地支即空亡。喜神空=吉力减半，忌神空=凶力减半，金空则鸣
- **D. 大运标注**：每步大运标注步数/干支/年龄/年份/十神/格局喜忌/调候喜忌
- **E. 调候与地理**：出生地五行方位辅助参考（北水/南火/东木/西金/中土）
- **F. 身强弱修正**：食神是药不是病，不应算作"克泄耗"削弱日主
- **G. 病药说**：四类病（刑冲破害/调候不宜/五行偏枯/格局有瑕），药即解决病的那个字或运

## 四、八字四家投票方法

四家并行 = ensemble 投票法（与机器学习随机森林同构）：

| 家 | 核心方法 | 强项 |
|----|---------|------|
| 徐乐吾 | 格局+调候 | 体系完整，子平真诠注释权威 |
| 梁湘润 | 三轨+金不换+四条规则 | 实务派，流月规则最精 |
| 袁树珊 | 命宫+小限+十六字法 | 看见别人看不见的暗损 |
| 韦千里 | 八步法+转角论 | 大运转角点定位精准 |

**置信度判定：**
- 四家一致 → 高置信度
- 三家一致 → 中置信度（记住少数派警告）
- 两两分歧 → 低置信度（等事实验证）

每家准确率约 70%，四家投票后共识结论准确率显著高于单家。

## 六、流月批导（必须两套并行）

### 梁湘润四条规则（流月只看与大运的关系）

1. **月运同甲子**（流月=大运伏吟）→ 大运效应集中爆发
2. **月运反吟**（流月冲大运）→ 动荡
3. **月运双合**（天干地支都合大运）→ 变质
4. **月运拱合**四柱+大运

流月与四柱无刑冲合会关系。流月与流年无刑冲合会关系。

### 袁树珊十六字法（流月看大运+小限+命宫）

小限推算：1岁小限 = 命宫干支，逐年逆行。

十六字 = 四柱 8 字 + 命宫 2 字 + 小限 2 字 + 大运 2 字 + 流年 2 字。

流月逐月对照大运（梁四条）+ 小限（冲合害伏吟）+ 命宫（冲合害伏吟），三层叠加。

### 两套结果对照

**两套结果必须对照，不一致时标注差异原因。**
